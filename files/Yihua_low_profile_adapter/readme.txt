One problem with the Jihua 858D is it occupies 2u of Ikea Samla. This modification fixes the problem and makes the Hot air gun compatible with 1u Ikea boxes.
This is done by replacing the original steel bracket with two printed parts which slide together and can be seperated for storage. 

Note: the STL file might be broken. In this case use the obj which is supported by most programs (e.g. Cura).

Fixes and restricts the minimal bend radius to enlong the cable life.

To install this part:
1. screw down the mount
2. screw on the top half
3. insert the cable
4. twist the top half to close the gap
5. ziptie the cables
6. if needed fix the top half by adding a pin

To remove the cables cut the zip tie and twist the top half to the open position.

Note: This part get's monted onto the PSU on the Anycubic Kossel so the "0016-2020-aluminium-extrusion-power-supply-(PSU)-mount-at-angle" PSU mount is required.

Remixed upgrade for standard extruder on 3D-printers to reduce gap/play to increase performance with soft fillaments like TPU.
Changes done are the added thread and numbering.

The original part was created by Atziano and can be found under Anycubic Kossel Extruder TPU Upgrade (https://www.thingiverse.com/thing:2762717). This part is licened under the Creative Commons - Attribution license.
